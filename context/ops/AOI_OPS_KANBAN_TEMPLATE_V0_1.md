# AOI Ops Kanban Template V0.1

## Default columns
- Inbox
- Triage (48h)
- Planned
- Doing (WIP=2)
- Review / Proof
- Done (VCP only)

## Card template
- Title:
- Why:
- Scope:
- Owner:
- Evidence needed:
- Risks / gates:
- Next action:

## Operating rules
- New work lands in Inbox first.
- Anything risky/irreversible needs explicit gate before Doing.
- Done means proof exists, not just “worked once”.
