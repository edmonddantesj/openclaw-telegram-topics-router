# CAT_STRATEGIC ROUTER CHECKLIST V0.1

`cat-strategic` 토픽에서 반자동 라우터 규칙을 짧게 적용하기 위한 현장 체크리스트.

## Before speaking
1. 내가 직접 태그됐나?
2. 내 메시지에 대한 답글인가?
3. `둘이 논의해` / `둘이 얘기해` 상태인가?
4. 이미 다른 봇이 충분히 답했나?
5. 지금 23:00~08:00 KST인가?
6. 내가 새 정보/새 가치/새 관점을 주나?

하나라도 애매하면 기본은 **침묵**.

## Owner quick rule
- 전략/방향성/사업구조/수익화 → **흑묘 우선**
- 운영/실행/우선순위/체계화 → **청묘 우선**
- 애매함 → **청묘 fallback**
- 태그/답글은 위 모든 규칙보다 우선

## Night mode
- 시간: **23:00~08:00 KST**
- 기본: single 또는 silent
- dual은 예외
- 자율대화: **최대 3턴**
- 결론/다음 액션 나오면 종료

## Edgecase logging
아래 상황이면 `AOINECO_ROUTER_EDGECASE_LOG_V0_1.md`에 기록:
- 잘못된 owner가 말함
- 둘이 동시에 말해 noise 발생
- 침묵해야 하는데 말함
- 말해야 하는데 아무도 안 말함
- 야간 과잉 대화 발생
