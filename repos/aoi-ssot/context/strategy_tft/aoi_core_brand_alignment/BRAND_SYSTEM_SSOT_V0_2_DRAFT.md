## 3) CHUNGHO Naming Rules (HARD)

### 3.1 Spelling / styling
- Official romanization: **CHUNGHO** (ALL CAPS)
- Never: ChungHo / Chung-ho / Cheongho / 청호(표기 변형) in external copy

### 3.2 First mention rule (strict)
- **First mention only:** **CHUNGHO (Blue Tiger)**
- **After first mention:** **CHUNGHO** only
- “Blue Tiger” is a gloss/meaning, **not the name**.

### 3.3 Placement rule
- CHUNGHO is a **motif / symbol name**, not a product SKU.
  - OK: “AOI Core — under the CHUNGHO symbol.”
  - Not OK: “CHUNGHO Core”, “CHUNGHO OS” (unless explicitly approved as a new product name).

## 4) Blue Bridge / 청교(靑橋) Usage Rules (HARD)

### 4.1 Word usage ban (default)
- **Do NOT overuse** the word “청교” in running text.
  - Risk: “청교도(Puritan)” unintended association.
- Default copy should prefer: **Bridge / Connection / Linking layers → outcomes**.

### 4.2 Allowed usage
- **Visual motif:** bridge / connection imagery (preferred)
- **Copy accent:** maximum **one line** (tagline-style), not repeated across the same page/deck.
- Treat as internal design philosophy: “we connect layers → outcomes”.

### 4.3 Disallowed usage
- Not a feature name, product name, or protocol name.
- Not used as a core keyword repeated in SEO paragraphs.
