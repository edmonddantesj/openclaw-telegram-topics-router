# Team Council — Pro Role Map v0.1 (Internal)

## 목적
Pro 버전에서 **우리 서버 내부(Aoineco)** 실행 시, 결과 보고서의 역할 이름을 **팀원 한글 닉네임**으로 표기하기 위한 고정 매핑표.

## 매핑 (Internal)
- 🧿 Oracle → 🧿 **청령**
- 🧠 Analyzer → 🧠 **청뇌**
- ⚔️ Security → ⚔️ **청검**
- ⚡ Builder → ⚡ **청섬**
- 📢 Comms → 📢 **청음**

## 규칙
- Lite(무료)는 항상 영문 역할명(+이모지)만 사용한다.
- Pro(유료)는 다음 우선순위로 표기한다:
  1) 내부(Aoineco 서버): 위 매핑(한글 닉네임)
  2) 외부 고객: 고객이 보유한 실제 에이전트명(워크포스 이름)을 그대로 사용
