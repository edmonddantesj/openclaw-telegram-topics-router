# AOI Squad Pro — Agent Ops Kit (Copy Pack) v0.1

## KR (10 lines)
1) AOI Squad Pro는 “에이전트 운영”을 제품으로 만드는 Proof‑first Ops OS다.
2) approve→run→proof 한 줄로, 실행 결과를 재현/감사 가능하게 고정한다.
3) 로그/아티팩트/해시(sha256)를 자동으로 묶어 ‘실행의 영수증(Proof Bundle)’을 만든다.
4) Shadow→Canary→Live로 위험을 단계적으로 통제한다.
5) L1/L2/L3 거버넌스가 내장되어 돈/보안/외부노출 작업은 무승인 실행을 막는다.
6) 크론이 돌아가도 “무엇이 왜 실패했는지” 3줄 요약+증빙 경로로 바로 복구한다.
7) Notion/로컬/Supabase SSOT를 연결해 운영 데이터가 흩어지지 않게 한다.
8) 단순 자동화가 아니라, 자동화의 ‘품질’과 ‘책임’을 올리는 운영 시스템이다.
9) 작은 자산($7 실험)부터 큰 자산까지 같은 규칙으로 확장 가능하다.
10) 목표: 더 빠르게, 더 안전하게, 더 증명 가능하게.

## EN (10 lines)
1) AOI Squad Pro is a proof‑first Agent Ops Kit — an Ops OS for running AI agents safely.
2) We standardize everything as approve → run → proof.
3) Every run produces a verifiable Proof Bundle (artifacts + logs + sha256).
4) Shadow → Canary → Live gates reduce blast radius.
5) Built‑in L1/L2/L3 governance prevents unapproved risky actions (funds/security/external).
6) When something fails, you get a 3‑line diagnosis + evidence paths — not noise.
7) SSOT sync keeps ops state consistent across local + Notion + Supabase.
8) Not “automation scripts” — operational quality, accountability, and reproducibility.
9) Designed to scale from micro‑bankroll experiments to serious production ops.
10) Faster. Safer. Verifiable.
