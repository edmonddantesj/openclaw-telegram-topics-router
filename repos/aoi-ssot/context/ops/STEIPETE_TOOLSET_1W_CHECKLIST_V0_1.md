# SteiPete Toolset 1주 운영 체크리스트 (v0.1)

## 목적
A/B 후보군 도입 후 1주 내 성과/리스크를 정량 점검해서 AOI Ops 다음 단계로 넘어갈지 결정.

## 기준 지표 (공통)
- 적용 툴명
- 설치/런타임 성공률 (성공회수/시도회수)
- 주당 시간절감 추정치 (분)
- 실패율
- 수동 개입 횟수
- 보안 이슈(토큰/권한/스크래핑 노출)

## 체크 항목
- [ ] 툴 설치 완료(버전 기록)
- [ ] 사용 워크플로에 템플릿 연동(샘플 1건)
- [ ] 증빙 캡처/로그 남김(최소 1개)
- [ ] 실패 케이스 1개 이상 재현 및 대응 절차 작성
- [ ] 보안 가드레일 위반 없음

## 정량 목표(1주)
- 리서치/분석 소요: -20% 이상
- 수동 보고/재요약 반복 감소: -15% 이상
- 툴 장애/재시도율: < 20%

## 결정 규칙
- 통과 조건: 3개 이상 지표 달성 + 보안 이슈 0
- 보류 조건: 설치 실패 반복 / 보안 우려 1건 이상 / 사용자 가치 미미(<10%)

## 다음 단계
- 통과: Planned로 승격 후 내재화(템플릿/Pipeline 반영)
- 보류: Security Review 재심의 또는 제외
