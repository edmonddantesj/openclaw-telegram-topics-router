# AOI Ops Kanban Template v0.1 (swarm-kanban)

## Board name
- AOI Ops

## Columns
1. Inbox
2. Triage (48h)
3. Planned
4. Doing (WIP=2)
5. Review / Proof
6. Done (VCP only)

## Rules
- **WIP=2**: Doing 컬럼엔 동시에 2개까지만(초과하면 Planned로).
- **Done 금지 조건**: URL/로그/파일경로 등 **증빙(VCP)** 없으면 Done으로 못 보냄.
- Triage는 48시간 내에: (a) 폐기 (b) Planned (c) 즉시 Doing 중 하나로 이동.
