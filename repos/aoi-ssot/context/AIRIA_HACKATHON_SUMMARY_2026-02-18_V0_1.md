# Airia AI Agents Hackathon — Summary v0.1 (2026-02-18)

Source: https://airia-hackathon.devpost.com/?utm_source=devpost&utm_medium=twitter&utm_campaign=airiaagents26#prizes

## What it is
Virtual hackathon to build AI agents for enterprise productivity. Two tracks.

## Tracks
1) **Airia Everywhere**
- Goal: meet users where they already are (reduce context switching)
- Target platforms (examples): browser extensions, Outlook/email, chat widgets, Slack/Microsoft Teams, SharePoint, WhatsApp, custom APIs

2) **Active Agents**
- Goal: multi-agent workflows across 2+ systems
- Capabilities highlighted: multi-system integrations, human-in-the-loop, dynamic document generation, nested agents, automated cross-platform workflows

## Required submissions
1) Agent name
2) Project description (problem, solution, features, tech used, target users)
3) Demo video (max **4 minutes**, YouTube/Vimeo, show real-time operation, narration/captions)
4) **Airia Community submission required** (must publish agent to Airia Community and include the community URL; agent must be public)

## Prizes (total $7,000 + credits)
Per track (Airia Everywhere / Active Agents):
- 1st: **$2,000** + $250 Airia credits
- 2nd: **$1,000** + $250 Airia credits
- 3rd: **$500** + $250 Airia credits
