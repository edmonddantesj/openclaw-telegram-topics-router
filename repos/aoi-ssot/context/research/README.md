# Research SSOT (Local) — context/research/

## 목적
- 개발 중 필요해진 주제/툴/패턴을 역할별로 빠르게 서칭하고, **요약+적용 포인트**를 남겨 팀 표준으로 축적한다.

## 저장 규칙
- 인박스(임시): `context/research/_inbox/`
- 템플릿: `context/research/_templates/RESEARCH_NOTE_TEMPLATE_V0_1.md`

## 작성 규격(최소 6줄)
1) 주제/키워드
2) 링크 1~5개
3) 한 줄 요약
4) 왜 지금 유용?
5) 우리 적용 포인트
6) 증빙/메모

## Notion 병행
- Notion Reference Inbox에도 동일 내용으로 기록(SSOT: cloud).
- 로컬은 빠른 검색/백업/리포지토리 증빙용.

## 파일 네이밍 권장
- `YYYY-MM-DD_<topic>_RESEARCH_NOTE_V0_1.md`
