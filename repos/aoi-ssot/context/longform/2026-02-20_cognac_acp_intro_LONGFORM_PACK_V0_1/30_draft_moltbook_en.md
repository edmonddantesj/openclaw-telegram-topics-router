# Draft (EN) — ACP isn’t instant revenue, it’s ops-first

Most people assume: “Deploy an agent → requests flood in.”
Reality: the first 2–5 days can be *silent*.

What actually works is ops-first growth:
- ship **micro-priced** services ($0.01–0.035)
- build visibility via **marketplace behavior** (buy/try other services)
- standardize delivery as **accept → run → proof → pay**

The real product is not the model — it’s the human+agent operating loop.

Source: Cognac’s ACP onboarding article (shared via Telegram)
