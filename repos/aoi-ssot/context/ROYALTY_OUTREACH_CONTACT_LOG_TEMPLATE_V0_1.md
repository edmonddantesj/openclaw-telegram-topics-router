# Royalty Outreach — Contact Log & Message Pack (V0.1)

## 1) Contact Attempt Log (one-line format)
Use **one row per attempt**.

**Format**
- `YYYY-MM-DD | skill/ref | creator | contact_method | address/handle | message_pack_version | result | proof`

**Fields**
- `skill/ref`: ClawHub skill name, GitHub repo, or internal ref id
- `creator`: name/handle
- `contact_method`: email / X / Discord / GitHub / website-form / 기타
- `address/handle`: 공개된 이메일/핸들/링크
- `message_pack_version`: 예) `PACK_V0_1`
- `result`: sent / delivered / no-reply / bounced / declined / agreed / follow-up
- `proof`: screenshot path, URL, or Notion block link

**Example**
- `2026-03-01 | stock-market-pro | @creator | email | hello@domain.com | PACK_V0_1 | sent | notion://...`

## 2) Pre-launch Message Pack checklist
(베타 오픈 2~3주 전에 일괄 전송용)
- Intro: Aoineco & Co. / Nexus Bazaar 소개
- What we used: 어떤 스킬/아이디어를 참고했는지(링크 포함)
- What we changed: 리빌딩 범위/차별점(요약)
- Attribution: 크레딧 표기 방식
- Royalty proposal: 옵션(%) / fixed / revshare 기준 / 정산 주기
- Ask: 연락 가능한 채널 + 동의/수정 요청
- Proof: 변경내역/SDNA/manifest/커밋/리포트 링크

## 3) Reminder rule (policy)
- **정식 런칭/베타 오픈 2~3주 전**: 위 Message Pack을 완성하고, Contact Attempt Log를 시작(최소 1회 이상 ‘연락 시도’ 증빙).
