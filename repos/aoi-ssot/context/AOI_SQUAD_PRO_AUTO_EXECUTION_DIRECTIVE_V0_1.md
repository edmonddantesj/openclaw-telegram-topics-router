# AOI Squad Pro 자동 실행 지시문 (v0.1)

**지시 즉시 발효(청묘/청령 공통):** 이후 항목은 별도 승인 대기 없이 L1/L2 범위에서 즉시 수행.

## 1) 실행 트리거(즉시)
- 청묘: 오늘 기준 2주 플랜 v1.0을 **Week 1 Day 1**부터 시작한다.
- 청령: 라운드 큐를 `RUNNING`에서 `COLLECTING`으로 전환하고 회수/재요청 루프 가동.

## 2) 청묘에게 할당(1회 공지)
```
AOI Squad Pro 2주 플랜(벤치마킹 소화형) 즉시 시행.
- L1/L2 범위는 승인 없이 자동 집행.
- 라운드 시작 알림만 발송 후 10분 내 템플릿 회신 수집.
- 필수 증빙 미기재는 DONE 처리 불가 (자동 BLOCK).
- L3 조건(자금 이동/키 노출/파괴적 작업) 발견 시 즉시 중단 후 보고.
```

## 3) 청령에게 할당(집계)
- 시작 시각/태스크셋/응답 1차 수집
- 미응답 2회 시 해당 슬롯 제외 및 대체 슬롯으로 넘김
- 공통 지표: 총 처리건, DONE/BLOCK/NEEDS_SUPPORT, 평균응답초, 증빙누락건수, BLOCK 사유

## 4) 2주 플랜 자동 집행 규칙
- Week1: 채택 구조 정착 + 증빙 스모크(10건, 4개 필수 증빙 100%) 진행
- Week2: 승인게이트 튜닝 + 예비 멤버 2명 파일럿 병행
- 종료일: 14일차 마지막 집계 리포트 자동 산출

## 5) 금지(즉시 차단)
- 새 기능이 L1/L2로 보이더라도, 외부 자금 이동/온체인 실시간 자동거래/지갑서명은 시작하지 않음.
- 토큰/재무/외부노출 변경은 L3 예외 라인.
