# Limitless Shadow Evidence Logging Protocol (v0.1)

Date: 2026-02-20 (KST)
Owner: Edmond + Aoineco
Status: READY

## Purpose
Ensure the Limitless hourly shadow experiment is continuously recorded with reproducible evidence bundles that can be reused later for:
- Hackathon submissions
- Investor due diligence
- Internal post-mortems and model iteration

## What we record (minimum viable evidence)
1) **Raw market odds snapshots (JSONL)**
- `/tmp/limitless_odds_btc_hourly.jsonl`

2) **Shadow settlements (JSONL)**
- `/tmp/limitless_settle_btc_hourly.jsonl`

3) **Model outputs (V6 archives)**
- `the-alpha-oracle/results/v6_*BTC-USD.json`

4) **Backtest results (odds-based)**
- Generated by: `scripts/v6_limitless_hourly_odds_backtest.py`

## Evidence bundle format
Daily (or on-demand) we create a proof bundle folder:
- `context/proof_samples/limitless_shadow_daily_<timestamp>/`
  - `counts.json` (line counts + coverage window)
  - `odds_tail.jsonl` (last N lines or last 24h window)
  - `settle_tail.jsonl`
  - `metrics.json` (ROI/bets/winrate/MDD + params)
  - `repro.txt` (exact commands used)

No secrets are included.

## Safety
- Report-only. Never executes real trades.
- Do not commit vault files or API keys.
- Public comms: no profit guarantees; present results as experiments with reproducible logs.

## Tools
- Hourly logger: `scripts/limitless_shadow_hourly_runner.sh`
- Daily snapshot packer: `scripts/limitless_shadow_daily_snapshot.py`
