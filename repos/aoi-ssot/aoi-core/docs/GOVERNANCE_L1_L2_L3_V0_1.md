# AOI Core Governance — L1/L2/L3 v0.1

## 목적
- L1/L2에서 **자동으로 최대한 전진**
- L3(의장) 결재가 필요한 건은 **즉시 브리핑**

## 용어 규칙 (MANDATORY)
- **"팀원"/"요원"** = Aoineco & Co. 내부 에이전트 10명만 의미
- 외부 인간/커뮤니티 계정/심사/업보터 등은 "외부 사용자/외부 참여자"로만 표현

## L1 (자율 실행 가능)
- 로컬 파일 생성/수정 (context/, aoi-core/, scripts/)
- 크론/파이프라인의 **출력 축약(safe_run)**, 로그 파일화
- 테스트/스모크/회귀테스트 추가
- 내부 토론 로그(로컬 JSONL) 축적

## L2 (참모진 실행 가능, 단 외부/SSOT 변경은 예외)
- 크론 job payload 개선(길이 제한/로그 경로 포함)
- 스킬 스캐폴드 생성/리팩
- 내부용 리서치(웹 검색) + 로컬 DB화(요약/링크/태그)

## L3 (의장 승인 필수)
- Notion SSOT에 **새 DB 생성/대량 업데이트**
- 외부 게시(봇마당/Moltbook/X) / ClawHub publish
- `main`에 직접 머지, 또는 배포/판매/가격정책 확정
- 시크릿/키/지갑 관련 정책 변경

## 브리핑 트리거(즉시 보고)
- 외부로 나갈 가능성이 있는 액션
- 금전/지갑/온체인 트랜잭션(라이브)
- 고객 영향(라이선스/환불/지원)
- 보안 리스크(토큰 노출 가능성)
